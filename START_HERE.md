# 🚀 GETTING STARTED - 5 MINUTES

## ⏱️ Quick Start (Choose Your OS)

### 🪟 Windows Users
```
1. Double-click install.bat
2. Wait for completion
3. Run: python Code/app.py
4. Open: http://localhost:5000
```

### 🍎 Mac Users
```
1. Open Terminal
2. Run: bash install.sh
3. Run: cd Code && python3 app.py
4. Open: http://localhost:5000
```

### 🐧 Linux Users
```
1. Open Terminal
2. Run: bash install.sh
3. Run: cd Code && python3 app.py
4. Open: http://localhost:5000
```

---

## 📊 What You Get

After running the app, you can:

1. **Upload Landsat Data**
   - 10 spectral bands (B1-B10)
   - MTL metadata file

2. **See Results**
   - Classification map
   - Area statistics
   - Download CSV

---

## 📥 Data Source

Download from: [USGS Earth Explorer](https://earthexplorer.usgs.gov/)

**What to download:**
- Landsat 8 Collection 2
- Level-1 data
- GeoTIFF format
